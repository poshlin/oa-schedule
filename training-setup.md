# 新人訓練頁配套設定指引

`training.html` 已上線，但有兩個外部資源需要你建立並把 URL 填回 HTML 才能完整運作：

1. **訓練頁問題回報 Form**（讓新人回報過期 / 不清楚的內容）
2. **師徒簽核 Master Sheet**（mentor 簽核 + 訓練內容維護表）

---

## 一、訓練頁問題回報 Google Form

### 建立步驟

1. 到 Google Form 新增空白表單，命名「橘子蘋果訓練頁問題回報」
2. 加入下列題目：

| 題號 | 題型 | 題目 | 必填 |
|------|------|------|------|
| 1 | 短答 | 你的英文名 | 是 |
| 2 | 短答 | 哪個章節（例：ch07） | 是（會用 URL 預填） |
| 3 | 多選一 | 問題類型 | 是 |
| 4 | 段落 | 詳細描述（哪裡不清楚 / 過期 / 連結壞了 ...） | 是 |
| 5 | 段落 | 建議怎麼改（選填） | 否 |

**第 3 題選項**：
- 內容過時 / 已不適用
- 連結壞掉 / 跑不到
- 文字寫得不清楚
- 缺少實際範例
- 跟其他資料矛盾
- 建議補充 / 新增內容
- 其他

**Form pre-fill 進階**：訓練頁的兩個按鈕（💡 建議改進、⚠️ 過期/錯誤）會帶 `entry.chapter` 和 `entry.type` 兩個參數。
- 「💡 建議改進」會帶 `entry.type=建議改進`
- 「⚠️ 過期/錯誤」會帶 `entry.type=過期或錯誤`

取得實際 entry id 的方法：
1. Form → 該題右上三個點 → 「取得預填連結」
2. 隨便填一個值，取得連結，複製
3. 把 HTML 內所有 `entry.chapter` / `entry.type` 換成你的真實 entry id（會是長數字）

不做也行，使用者進 Form 後手動選即可。

### 取得 URL

建好後，點右上「傳送」→「連結」→ 複製 URL（會像 `https://docs.google.com/forms/d/e/1FAIp.../viewform`）。

### 填入 HTML

打開 `training.html`，找到第 ~350 行：
```js
const REPORT_FORM_URL = "https://docs.google.com/forms/d/e/REPLACE_WITH_YOUR_FORM_ID/viewform";
```
把後半段換成你的 Form URL。

### 進階：URL 預填章節 id（推薦）

頁面上每個「⚠️ 這節有問題」按鈕會自動帶 `?entry.chapter=ch07` 這種參數。
要讓 Form 接到這個 entry，需要：

1. 開啟 Form → 點該題（章節題）右上三個點 → 「取得預填連結」
2. 隨便填一個值（例：`ch07`）→ 取得連結 → 複製
3. 連結會像 `...?usp=pp_url&entry.1234567890=ch07`
4. 把 HTML 內所有 `entry.chapter` 換成 `entry.1234567890`（你的真實 entry id）

不做也行，新人就要自己手動選章節。

### 填入 HTML

打開 `training.html`，找到第 ~351 行：
```js
const MENTOR_SHEET_URL = "https://docs.google.com/spreadsheets/d/REPLACE_WITH_YOUR_SHEET_ID/edit";
```

### 權限設定

- **誰可編輯**：你 + 全部 mentor（top sales）+ 課務主管
- **誰可瀏覽**：上述人員之外都不開放
- **新人看不到 Sheet**（他在 `training.html` 內勾 checklist 即可，那是個人 localStorage）
